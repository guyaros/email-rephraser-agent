# Email Rephraser Agent

An AI-powered tool for rephrasing professional support emails to equipment suppliers.

## What it does
Takes rough email drafts, bullet points, or notes and rephrases them into clear, professional emails tailored to supplier support or other teams. 
The agent considers device details, urgency level, conversation context, and previous supplier messages to generate a relevant and coherent reply.

## Key features
- Device model selector (Formlabs and Markforged devices)
- Urgency levels: Low, Normal, High (production blocker)
- Tone and goal selection
- Optional context, actions taken, and previous message fields
- Outputs a subject line and email body
- Editable output with "Improve this version" option

## Tech stack
- Pure HTML / CSS / JavaScript (single file)
- Anthropic Claude API (claude-sonnet)
- Hosted on GitHub Pages
